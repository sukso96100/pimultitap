# rpi-relay-control
Node.js based relay module power control panel for Raspberry Pi

Start without debug
```bash
npm start
```
for debug, run the following command.
```bash
DEBUG=rpi-relay-control:* npm start
```

### Requirements
- 8 Channel Relay Module for your RPi
- SQLite3
- Node.js and Node.js Modules
  - forever
  - express
  - sqlite3
  - onoff
  - body-parser
  - cookie-parser
  - debug
  - ejs
  - morgan
  - serve-favicon
